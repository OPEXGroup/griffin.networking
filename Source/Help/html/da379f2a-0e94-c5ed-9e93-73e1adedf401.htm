Server & Client implementation that can transfer messages (POCOs) over the network.

Service which can send and receive messages.

Interfaces

  InterfaceDescription
IMessageBuilder
Used to build a message from buffers
IMessageFormatterFactory
The following factory are used to provide the classes that are used to convert your .NET POCOS to something that can be transported over a socket and vice versa.
IMessageSerializer
Serializes the message into the writer

Remarks

Do note that the actual format of the transfers are not defined in this namespace. You can however install the griffin.networking.basic nuget package to get a messaging implementation that uses a small binary header (version+length) and JSON to transport all pocos.
