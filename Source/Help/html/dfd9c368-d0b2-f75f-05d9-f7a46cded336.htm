Remarks

Do note that it's recommended that you pool buffers (i.e.) reuse them. You can use the BufferSliceStack for that.

All buffers that are reusable should implement IDisposable. Buffers are returned to the pool by calling Dispose(). You therefore have to check each buffer if it's disposable when you are done with it (and dispose it if it is).
